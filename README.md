
## 🚀 Running Locally

1. Clone this repository:
   ```bash
   git clone https://learn.reboot01.com/git/ssudqi/graphql.git
   ```
2. Open the project folder:
   ```bash
   cd graphql
   ```
3. Open `login.html` in your browser to get started.
